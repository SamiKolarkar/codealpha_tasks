1.The above java application is a CMD based java application 
  for tracking student grades.
2.It takes input from user as 'comma-separated' string and
  converts it into array of grades as double datatype.
3.To calculate average,highest and lowest;3 variables as 
  sum,avg,highest and lowest are used.
4.By traversing records array each is checked and the corresponding
  values required are updated.
5.The final value contained in those variables are the required
  details for tracking student grades,just sum is divided by 
  total grade records to find average.
6.The grades are printed one by one as per given order separated
  by commas.
